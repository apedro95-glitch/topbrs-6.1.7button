# SISTEMA AUTOMÁTICO TOP BRS' 🇧🇷 — V6.0 Firebase Realtime

Esta versão está preparada para usar Firebase + Cloud Firestore em tempo real.

## Arquivos novos
- `firebase-config.js` → preencha com os dados do seu projeto Firebase
- `firebase-config.example.js` → modelo de referência
- `firebase-sync.js` → sincronização em tempo real + login do líder

## Como funciona
- Sem Firebase configurado: roda local igual antes
- Com Firebase configurado: todos veem os dados em tempo real
- Quem entra com o email do líder pode editar e salvar online
- O clã abre o mesmo link em modo leitura
